# b2bOptic Lens Order Interface

## Install:
npm install b2boptic_lensorder

## Usage:
var b2bOptic_lensorder = require('b2boptic_lensorder');  
var b2bOptic_lensorder_json_object = b2bOptic_lensorder.b2bOptic();

## Next release:
